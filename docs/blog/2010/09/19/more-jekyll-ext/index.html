more jekyll ext

19 Sep 2010 on jekyll

昨日に引き続きjekyll_extのお試し。

my_jekyll_extensions の中から

  • archive_gen
    • 年／月／日毎にアーカイブページを生成する
    • 年 (_layout/archive_yearly.html)
    • 月 (_layout/archive_monthly.html)
    • 日 (_layout/archive_daily.html)
  • tag_gen
    • 記事にタグが付けられる
      • tags: hoge foo bar のようにスペース区切りで
    • tag page (_layouts/tag_index.html)
    • tag cloud (tags/index.html)
  • categories_in_posts
    • _posts/ 以下にディレクトリを作成することでカテゴリー分けすることができる(未使用)
  • category_gen
    • カテゴリー別のページを生成する
    • まだ記事にカテゴリー付けてないのでサンプル無し
    • category list (categories/index.html)
  • tag_category_iterator
    • site.tags や site.categories が使えるようになる

これでだいぶブログっぽくなってきた。
